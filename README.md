# Numera

Site oficial da Numera — empresa de tecnologia financeira que desenvolve aplicações, metodologias e infraestrutura para análise e gestão financeira inteligente.

**Acesse:** [numera.com.br](https://numera.com.br)

## Sobre

O site consolida o ecossistema de produtos Numera: aplicações web, cases de sucesso, métricas de impacto e a metodologia proprietária da empresa — servindo como vitrine institucional e ponto de entrada para todos os produtos.

## Seções

- **Hero** — Apresentação institucional
- **Exchange Ticker** — Cotações de câmbio em tempo real
- **Infraestrutura** — Diferenciais tecnológicos
- **Métricas** — Indicadores de impacto e escala
- **Metodologia** — Abordagem e filosofia
- **Ecossistema** — Portfólio de aplicações
- **CTA** — Chamada para ação

## Funcionalidades

- Suporte a múltiplos idiomas com Context API de i18n
- Ticker de câmbio com dados ao vivo
- Efeito de cursor spotlight
- Design glassmorphism responsivo
- Animações com Framer Motion

## Stack

- **Frontend:** React 19 + Vite + Tailwind CSS 4
- **Animações:** Framer Motion
- **i18n:** Context API

## Instalação

```bash
git clone https://github.com/seu-usuario/numera.git
cd numera
npm install
npm run dev
```

## SEO

Inclui `public/sitemap.xml`. Submeta no [Google Search Console](https://search.google.com/search-console) após publicar. Atualize o domínio no arquivo antes do deploy.
